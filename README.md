# newLandPage

Simple Landing Page mockup built with **HTML**, **CSS**, and **JavaScript**.for LINEA package tracker SaaS.

## Preview / Pages
- `index.html` (main landing page)
- `about.html` (secondary page)

## Tech
- HTML / CSS / JavaScript
- External libs via CDN (in `index.html`): GSAP, Swiper, Swup, Iconify

Optional (recommended): use a local server (example with VS Code **Live Server**).

## Project structure
- `index.html`
- `about.html`
- `css/` (styles)
- `js/` (scripts)
- `assets/` (images and other static files)
